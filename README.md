# 💻 Laptop Price Prediction using Machine Learning

A Machine Learning project that predicts laptop prices based on their specifications. This project demonstrates the complete data science workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation using Python.

---

## 📌 Project Overview

Laptop prices depend on various hardware and software specifications such as brand, processor, RAM, storage, GPU, operating system, screen resolution, and display size. The objective of this project is to build a regression model capable of accurately predicting laptop prices based on these features.

---

## 🎯 Objectives

- Analyze the factors affecting laptop prices.
- Clean and preprocess raw data.
- Perform Exploratory Data Analysis (EDA).
- Engineer relevant features for model training.
- Train and evaluate Machine Learning regression models.
- Predict laptop prices with improved accuracy.

## 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/your-username/Laptop-Price-Prediction.git
```

2. Navigate to the project directory

```bash
cd Laptop-Price-Prediction
```

3. Install the required libraries

```bash
pip install -r requirements.txt
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open

```
laptop_price_analysis.ipynb
```

and run all cells.
